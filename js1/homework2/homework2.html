Задание 1
Дан код:
var a = 1, b = 1, c, d;
1) c = ++a; alert(c); // 2
2) d = b++; alert(d); // 1
3) c = (2+ ++a); alert(c); // 5
4) d = (2+ b++); alert(d); // 4
5) alert(a); // 3
6) alert(b); // 3
Почему код даёт именно такие результаты?
Решение:
1) Возвращается значение, полученное после выполнения инкремента: 1 + 1 = 2
2) Инкремент постфикс - то, что было до инкремента: 1
3) Сначала инкремент префикс, потом остальное: 2 + ((a=2)+1) присвоить c
4) Сложение с предыдущим значением: 2 + 2 = 4 присвоить d
5) a увеличивалось в предыдущих операциях инкремента 2 раза на 1
6) b увеличивалось в предыдущих операциях инкремента 2 раза на 1

var a = 2;

Решение:
x = 1 + (a = a * 2 = 4) = 5
